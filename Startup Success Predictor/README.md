# Startup Success Predictor

## Project Overview

This project uses startup and venture capital investment data to build models for predicting startup success.

## Files

- `startup_success_predictor.ipynb` — main analysis notebook.
- `start-up-success-classification-streamlit-app.ipynb` — Streamlit app prototype for model interaction.
- `investments_VC.csv` — dataset used for training and evaluation.

## How to run

1. Open `startup_success_predictor.ipynb` or `start-up-success-classification-streamlit-app.ipynb` in Jupyter or VS Code.
2. Ensure `investments_VC.csv` is in the same folder.
3. Run the notebook cells from top to bottom.

## Requirements

- Python 3.x
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn
