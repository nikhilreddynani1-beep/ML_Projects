# Customer Personality Analysis

## Project Overview

This project analyzes customer personality and marketing campaign data to identify patterns in behavior and customer segments.

## Files

- `Customer Personality Analysis.ipynb` — main notebook with exploratory data analysis and modeling.
- `marketing_campaign.csv` — dataset used for the analysis.
- `eda_distributions.png` — exploratory visualization output.
- `eda_pca.png` — principal component analysis visualization.

## How to run

1. Open `Customer Personality Analysis.ipynb` in Jupyter or VS Code.
2. Ensure `marketing_campaign.csv` is in the same folder.
3. Run the notebook cells from top to bottom.

## Requirements

- Python 3.x
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn
